La etiqueta "big" no está incluida en la lista de etiquetas HTML5 porque se considera obsoleta y su uso se desaconseja. En su lugar, se recomienda utilizar CSS para establecer el tamaño del texto y otros estilos visuales.

Por otro lado, la etiqueta "small" aún está incluida en HTML5, ya que se considera útil para indicar que un fragmento de texto es menos importante o tiene un tamaño menor que el texto circundante. Sin embargo, también se recomienda el uso de CSS para establecer el tamaño del texto y otros estilos visuales, en lugar de depender de la etiqueta "small".

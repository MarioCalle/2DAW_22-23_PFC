Lenguajes de marcas y sus tipos

Lenguaje de marca Tipo Descripción
HTML De presentación Permite la creación de páginas web con diferentes elementos visuales.
CSS De presentación Se utiliza para dar estilo y formato a los elementos creados con HTML.
XML De datos Se utiliza para almacenar y transportar datos, y es compatible con diferentes plataformas.
JSON De datos Se utiliza para transmitir datos entre diferentes sistemas y aplicaciones.
YAML De datos Es un formato de serialización de datos que se utiliza en diferentes lenguajes de programación.
